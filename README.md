# Jungler
League of Legends and Teamfight Tactics companion app for Fitbit Watches. Currently includes 5 minute timers for summoner spell cooldowns. 

## Building the App
This app was produced with Fitbit Studio. Import the project to Fitbit Studio or build the project with the CLI tools.

## Planned features
- Item suggestions based on Summoner statistics.
- Competitor statistics at a glance.
